# tidal-share

Tidal feature.fm share link viewer (songs only)

Works without JS and on slow internet connections

Examples:
<https://tidal.com/track/58990486/u> -> <https://tidal.tpgy.uk/track/58990486/u>
<https://tidal.com/track/451263785/u> -> <https://tidal.tpgy.uk/https://tidal.com/track/451263785/u>
<https://tidal.com/track/424363595/u> -> <https://tidal.tpgy.uk/424363595>
